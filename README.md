# kenshincheckup

> [!NOTE]
> This project is under development. See DESIGN.md for the current specification.

## Overview

kenshincheckup is a periodic health-check tool for desktop OS environments.

## Usage

```
kenshincheckup
kenshincheckup help
kenshincheckup --help
kenshincheckup -h
kenshincheckup --config <path>
kenshincheckup -c <path>
```

By default, the config is read from `~/.config/kenshin/config.toml`.
